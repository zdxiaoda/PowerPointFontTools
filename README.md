# PowerPointFontTools (PPT 字体工具)

一个用于分析 PowerPoint (.pptx) 文件中所使用字体的实用工具。它可以帮助你快速识别演示文稿中缺失的字体，并允许你将已安装的字体导出为 ZIP 包，以便在其他设备上部署。

## 主要功能

- **深度分析**: 扫描 PPT 幻灯片内容，精确提取所有使用的字体族名称。
- **本地字体数据库**: 启动时自动扫描系统和用户目录下的字体文件，建立高效的本地索引（`fonts.db`）。
- **缺失字体识别**: 自动对比 PPT 中的字体与本地已安装字体，直观显示“已安装”和“缺失”列表。
- **字体导出**:
    - 一键导出缺失字体名单 (`.txt`)。
    - 一键将 PPT 中用到的已安装字体打包成 ZIP 文件，方便打包交付。
- **快速搜索**: 双击列表中的缺失字体，直接在浏览器中搜索下载链接。
- **智能匹配**: 支持多种字体名称变体匹配（如带空格、连字符、下划线等），解决文件名与显示名称不一致的问题。

## 如何使用

1. **资源初始化**: 首次运行程序会扫描系统字体目录，这可能需要几秒钟。完成后会生成本地数据库供后续快速复用。
2. **导入文件**: 直接将 `.pptx` 文件拖入程序的灰色放置区。
3. **查看结果**:
    - **左侧列表**: 显示当前系统缺失的字体。双击即可打开浏览器搜索。
    - **右侧列表**: 显示 PPT 中已使用且本地已安装的字体。
4. **导出**:
    - 点击“导出缺失字体名单”保存为文本。
    - 点击“导出已安装字体”将字体原文件打包导出。
5. **维护**: 如果你安装了新字体，可以点击“重新加载字体库”更新本地索引。
